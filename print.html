Membuat Service Type

A. INPUT

(Tidak ada instruksi khusus)

B. LANGKAH KERJA

  1. Buka menu Service -> Configuration -> Service Type. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Klik tombol Create pada bagian atas-kiri form.

  1. Isi Type. Wajib diisi.
  2. Isi Code. Wajib diisi.
  3. Aktifkan/ Deaktifkan Active. Tidak wajib diisi.
  4. Pilih Contract Sequence. Tidak wajib diisi.
  5. Pilih Custom Info Template. Tidak wajib diisi.
  6. Beralih ke tab Fix Items Configuration.
  7. Pilih Receivable Journal. Tidak wajib diisi.
  8. Pilih Receivable Account. Tidak wajib diisi.
  9. Pilih Allowed Products. Wajib diisi.
  10. Pilih Allowed Product Categories. Tidak wajib diisi.
  11. Beralih ke tab Team.
  12. Pilih Allowed Products. Wajib diisi.
  13. Pilih Allowed Product Categories. Tidak wajib diisi.
  14. Beralih ke tab Analytic & Project.
  15. Pilih Parent Analytic Account. Tidak wajib diisi.
  16. Aktifkan/ Deaktifkan Auto Create Project. Tidak wajib diisi.
  17. Beralih ke tab Workflow Policy.
  18. Pilih Allow To Confirm Contract. Tidak wajib diisi.
  19. Pilih Allow To Restart Approval Contract. Tidak wajib diisi.
  20. Pilih Allow To Force Start Contract. Tidak wajib diisi.
  21. Pilih Allow To Force Finish Contract. Tidak wajib diisi.
  22. Pilih Allow To Terminate Contract. Tidak wajib diisi.
  23. Pilih Allow To Cancel Contract. Tidak wajib diisi.
  24. Pilih Allow To Restart Contract. Tidak wajib diisi.
  25. Beralih ke tab Service Quotation Configuration.
  26. Pilih Quotation Sequence. Tidak wajib diisi.
  27. Pilih Quotation Custom Info Template. Tidak wajib diisi.
  28. Pilih Allow To Confirm Service Quotation. Tidak wajib diisi.
  29. Pilih Allow To Restart Approval Service Quotation. Tidak wajib diisi.
  30. Pilih Allow To Mark As Won Service Quotation. Tidak wajib diisi.
  31. Pilih Allow To Mark As Lost Service Quotation. Tidak wajib diisi.
  32. Pilih Allow To Cancel Service Quotation. Tidak wajib diisi.
  33. Pilih Allow To Restart Service Quotation. Tidak wajib diisi.
  34. Aktifkan/ Deaktifkan Required Start Date. Tidak wajib diisi.
  35. Aktifkan/ Deaktifkan Required End Date. Tidak wajib diisi.
  36. Beralih ke tab Note.
  37. Isi Note. Tidak wajib diisi.
  38. Beralih ke tab Operating Unit.
  39. Pilih Operating Unit. Tidak wajib diisi.
  40. Pilih Contract Sequence. Tidak wajib diisi.
  41. Pilih Quotation Sequence. Tidak wajib diisi.
  42. Pilih Fix Item Receivable Journal. Tidak wajib diisi.
  43. Pilih Fix Item Receivable Account. Tidak wajib diisi.
  44. Klik tombol Save pada bagian atas-kiri form.

Menghapus Service Type

A. INPUT

  • User yang akan menghapus harus memiliki akses untuk menghapus Service Type.

B. LANGKAH KERJA

  1. Buka menu Service -> Configuration -> Service Type. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data Service Type yang akan dihapus. Abaikan jika data sudah dibuka.
  3. Klik tombol More pada bagian atas-tengah form.

  1. Klik tombol Delete pada dropdown yang muncul ketika tombol More diklik.

  1. Klik tombol Ok pada pop-up konfirmasi penghapusan yang muncul.

C. OUTPUT

  • Data Service Type akan terhapus.

Menonaktifkan Service Type

A. INPUT

  • Data Service Type yang akan dinonaktifkan harus memiliki status Active

B. LANGKAH KERJA

  1. Buka menu Service -> Configuration -> Service Type. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data Service Type yang akan dinonaktifkan. Abaikan jika data sudah dibuka.
  3. Klik tombol Edit pada bagian atas-kiri form.

  1. Deaktifkan Active.

  2. Klik tombol Save pada bagian atas-kiri form.

C. OUTPUT

  • Data Service Type akan non aktif.
